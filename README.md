# 물류 입출력 시뮬레이션

웹·모바일에서 동작하는 **물류 시뮬레이션** 프로젝트입니다.  
장비 배치, 물품 이동 속도, 처리 속도, 전력 등을 고려한 시뮬레이션을 지원합니다.

- **배치·테스트**: 그리드 위에 장비를 배치하고 동작을 테스트
- **시뮬레이션**: 시간에 따른 흐름·처리량·전력 등을 분석

---

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173` 접속 후, 상단에서 **배치·테스트** / **시뮬레이션** 모드를 선택해 사용합니다.
